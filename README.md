# fate_app

## Описание проекта

fate_app - это приложение на Flutter, разработанное для управления персонажами в ролевых играх. Оно предоставляет пользователям возможность создавать, редактировать и просматривать персонажей, а также управлять их характеристиками, навыками и другими аспектами.

## Основные функции

- **Создание персонажей**: Пользователи могут добавлять новых персонажей с уникальными характеристиками.
- **Редактирование персонажей**: Возможность изменять информацию о персонаже, включая имя, концепцию и навыки.
- **Просмотр информации**: Удобный интерфейс для просмотра всех деталей персонажа, включая его аспекты и трюки.
- **Генерация PDF**: Создание PDF-документов с информацией о персонаже для удобного хранения и печати.
- **Интерактивные элементы**: Использование выпадающих меню и текстовых полей для ввода данных.

## Начало работы

Чтобы начать работу с проектом, выполните следующие шаги:

1. **Клонируйте репозиторий**:
   ```bash
   git clone <URL_репозитория>
   cd fate_app
   ```

2. **Установите зависимости**:
   ```bash
   flutter pub get
   ```

3. **Запустите приложение**:
   ```bash
   flutter run
   ```
